แนะนำ Power Platform และเริ่มต้นกับ Power Apps

ยินดีต้อนรับสู่บทเรียนแรก! ในบทนี้ เราจะสำรวจโลกของ Power Platform และเรียนรู้วิธีสร้างแอปพลิเคชันแรกของคุณด้วย Power Apps ตั้งแต่เริ่มต้น

Power Platform คืออะไร?

คือชุดเครื่องมือ Low-Code จาก Microsoft ที่ช่วยให้ทุกคนสร้างโซลูชันทางธุรกิจได้อย่างรวดเร็ว ประกอบด้วยองค์ประกอบหลัก 4 อย่าง

Power Apps

เครื่องมือสำหรับสร้างแอปพลิเคชันทางธุรกิจที่กำหนดเองได้โดยไม่ต้องเขียนโค้ดซับซ้อน

Power Automate

เครื่องมือสำหรับสร้าง Workflow อัตโนมัติเพื่อเชื่อมโยงแอปและบริการต่างๆ เข้าด้วยกัน

Power BI

เครื่องมือสำหรับวิเคราะห์ข้อมูลและสร้าง Dashboard เพื่อแสดงผลข้อมูลเชิงลึก

Power Virtual Agents

เครื่องมือสำหรับสร้าง Chatbot อัจฉริยะเพื่อตอบคำถามและให้บริการลูกค้า

คำอธิบายของส่วนต่างๆ ใน Power Apps Studio จะแสดงที่นี่

ภาคปฏิบัติ: สร้างแอปแรกของคุณ

1. ไปที่ make.powerapps.com

2. คลิก Create > Canvas app from blank

3. ตั้งชื่อแอปว่า "MyFirstApp" และเลือกรูปแบบ Tablet

4. กด Create

1. ที่แถบ Ribbon ด้านบน คลิก Insert > Label

2. ที่ Properties Pane ด้านขวา เปลี่ยนค่า Text เป็น "ยินดีต้อนรับ!"

3. คลิก Insert > Button

4. เปลี่ยนค่า Text ของ Button เป็น "ไปต่อ"

1. ที่ Tree View (ด้านซ้าย) คลิก New screen > Blank เพื่อสร้างหน้าจอที่สอง

2. กลับมาที่หน้าจอแรก และเลือกปุ่ม "ไปต่อ"

3. ที่ Formula Bar เลือก Property เป็น OnSelect

4. พิมพ์สูตร:

Navigate(Screen2, ScreenTransition.Fade)

5. กดปุ่ม Play (▶) ที่มุมขวาบนเพื่อทดสอบการทำงาน

หลักการของ Formula: หัวใจของการสร้างแอป

สูตรใน Power Apps มีลักษณะคล้ายกับการเขียนสูตรใน Excel แต่ใช้เพื่อกำหนด "การกระทำ" (Action) และ "คุณสมบัติ" (Property) ของส่วนประกอบต่างๆ ในแอป

Formula คืออะไร?

คือชุดคำสั่งที่เราใช้ในการบอก Power Apps ว่าเมื่อเกิดเหตุการณ์บางอย่างขึ้น (เช่น ผู้ใช้กดปุ่ม) แอปจะต้องทำอะไรต่อ

ตัวอย่าง: การเปลี่ยนหน้าจอด้วยสูตร Navigate()

สูตรนี้ใช้สำหรับการนำทางจากหน้าจอหนึ่งไปยังอีกหน้าจอหนึ่ง เมื่อคุณเขียนสูตรนี้ใน Property ชื่อ OnSelect ของปุ่ม

ลองดูตัวอย่างสูตร `Navigate()` นี้

Navigate(Screen2, ScreenTransition.Fade)

เมื่อคุณกดปุ่มด้านล่าง แอปจะเปลี่ยนไปหน้าจอที่ 2 โดยใช้เอฟเฟกต์ Fade

*(ปุ่มนี้เป็นตัวอย่าง ไม่มีผลจริง)*

Workshop (1 ชั่วโมง): สร้างแอป Task Manager

มาลงมือทำจริง! สร้างแอปจัดการงานอย่างง่าย เพื่อทบทวนความรู้ทั้งหมดที่เรียนมาในวันนี้

1. สร้าง Canvas App ใหม่ในรูปแบบ Tablet

2. สร้าง 2 หน้าจอ:

  • เปลี่ยนชื่อ Screen1 เป็น HomeScreen
  • สร้างหน้าจอใหม่และตั้งชื่อว่า NewTaskScreen

บนหน้า HomeScreen:

  • เพิ่ม Label ที่มีข้อความว่า "รายการงานของฉัน"
  • เพิ่ม Button ที่มีข้อความว่า "เพิ่มงานใหม่"
  • เขียนสูตรที่ปุ่มนี้ให้ Navigate ไปยัง NewTaskScreen

บนหน้า NewTaskScreen:

  • เพิ่ม Label "ชื่องาน" และ Text Input สำหรับกรอกชื่องาน
  • เพิ่ม Button "บันทึก" และ "ยกเลิก"
  • เขียนสูตรที่ปุ่ม "ยกเลิก" ให้ Navigate กลับไปยัง HomeScreen

1. กดปุ่ม Play เพื่อทดสอบการทำงานของแอป

2. ตรวจสอบว่าปุ่ม "เพิ่มงานใหม่" และ "ยกเลิก" ทำงานถูกต้องหรือไม่

3. เตรียมคำถามที่สงสัยไว้ถามผู้สอน

เยี่ยมมาก! คุณได้สร้างพื้นฐานของแอป Task Manager สำเร็จแล้ว!
