ยินดีต้อนรับสู่ Power Apps

เริ่มต้นสู่โลกของ Low-Code! ในบทนี้ เราจะมาเรียนรู้การสร้างแอปพลิเคชันด้วย Power Apps โดยไม่ต้องเขียนโค้ดที่ซับซ้อน

Power Apps คืออะไร?

Power Apps คือเครื่องมือที่ช่วยให้เราสร้างแอปพลิเคชันสำหรับใช้งานบนเว็บไซต์และโทรศัพท์มือถือได้อย่างรวดเร็ว โดยอาศัยแนวคิด Low-Code/No-Code

หน้าจอ (Interface) ของ Power Apps

หน้าจอหลักของ Power Apps Studio มี 3 ส่วนสำคัญคือ Tree View, Canvas, และ Properties Panel ซึ่งคล้ายกับการสร้างสไลด์ใน PowerPoint แต่มีพลังของสูตรและข้อมูลที่อยู่เบื้องหลัง

คลิกที่หมายเลขเพื่อเริ่มต้น

คำอธิบายของส่วนต่างๆ ใน Power Apps Studio จะแสดงที่นี่

การ Insert ทุกอย่าง (Insert Controls)

Controls คือส่วนประกอบต่างๆ ที่เราใช้ในการสร้างแอป Tab Insert มีปุ่มสำหรับเพิ่มคอนโทรลมากมาย เช่น Label, Text Input, Button, Dropdown, Gallery, Forms

ตัวอย่างสอน:

  • Insert → Label → เปลี่ยนข้อความเป็น “ชื่อผู้ใช้”
  • Insert → Text Input → ใส่ค่าเริ่มต้น = “กรอกชื่อ”
  • Insert → Button → ตั้งข้อความ = “บันทึก”
  • Insert → Dropdown → ตัวเลือก (ชาย/หญิง)
  • Insert → Gallery → แสดงรายการนักศึกษา

การใช้ตารางจาก Excel / การสร้าง Collection

เราสามารถดึงข้อมูลจากไฟล์ Excel มาใช้ หรือสร้างตารางชั่วคราวในแอปด้วย Collection เพื่อเก็บค่าที่ผู้ใช้ป้อนก่อนบันทึกจริง

ตัวอย่างสอน:

  • เชื่อม Excel “Student.xlsx” ตาราง StudentList → แสดงใน Gallery
  • ใส่ปุ่ม “Save” → Collect ข้อมูลจาก Text Input ลง Collection StudentCol

การสร้าง Action พื้นฐาน

Action คือสิ่งที่เกิดขึ้นเมื่อผู้ใช้กดหรือเลือกคอนโทรล ซึ่งเราจะกำหนดด้วยสูตรง่าย ๆ ที่เห็นผลทันที

ตัวอย่างสอน:

  • ปุ่ม Next → `Maps(Screen2, ScreenTransition.Fade)`
  • ปุ่ม Save → `Collect(StudentCol, {Name: TextInput1.Text})`
  • ปุ่ม Clear → `Reset(TextInput1)`
  • ปุ่ม Exit → `Exit()`
  • ปุ่ม Show/Hide → `UpdateContext({showPanel: !showPanel})`

สรุปบทเรียน

Power Apps คือเครื่องมือสร้างแอปง่าย ๆ ที่ส่วนติดต่อเหมือน PowerPoint แต่มีสูตรคุมได้ เริ่มจาก Insert คอนโทรล → เชื่อม Excel/สร้าง Collection → ใส่ Action
